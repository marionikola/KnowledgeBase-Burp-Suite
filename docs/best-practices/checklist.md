# Pre-Engagement Checklist - Burp Suite

## Pre-Engagement

### Authorization
- [ ] Written scope document
- [ ] Legal agreement/NDA
- [ ] Emergency contact list
- [ ] Client signature on scope

### Technical Scope
- [ ] Target URLs defined
- [ ] Out-of-scope defined
- [ ] Subdomains in scope?
- [ ] APIs in scope?
- [ ] Mobile apps included?
- [ ] Third-party integrations?

### Timeline
- [ ] Start date agreed
- [ ] End date agreed
- [ ] Working hours defined
- [ ] Buffer for delays

---

## Setup Phase

### Tools Preparation
- [ ] Burp Suite configured
- [ ] Proxy listener working
- [ ] CA certificate installed
- [ ] Browser configured
- [ ] Project saved

### Documentation
- [ ] Project file created
- [ ] Scope added to Target tab
- [ ] Notes section ready

---

## Reconnaissance

### Passive Information Gathering
- [ ] WHOIS lookup
- [ ] DNS enumeration
- [ ] Public documents reviewed
- [ ] Job postings checked

### Active Reconnaissance
- [ ] Port scan completed
- [ ] Technology detection
- [ ] Subdomain enumeration

---

## Mapping Phase

### Crawling
- [ ] Spider completed
- [ ] All pages discovered
- [ ] Forms identified
- [ ] Hidden directories found

### Manual Exploration
- [ ] All features tested
- [ ] User flows documented
- [ ] Entry points identified

---

## Testing Phase

### Authentication Testing
- [ ] Login functionality
- [ ] Logout functionality
- [ ] Password reset
- [ ] Session management
- [ ] MFA if applicable

### Authorization Testing
- [ ] IDOR checks
- [ ] Privilege escalation
- [ ] Access control

### Input Validation
- [ ] SQL Injection
- [ ] XSS (Reflected, Stored, DOM)
- [ ] Command Injection
- [ ] Path Traversal
- [ ] XXE

### Session Management
- [ ] Session tokens
- [ ] Cookie attributes
- [ ] Session fixation
- [ ] Logout behavior

---

## Reporting

### Finding Documentation
- [ ] All findings verified
- [ ] Evidence captured
- [ ] Screenshots taken
- [ ] Steps documented

### Quality Check
- [ ] No false positives
- [ ] Proper severity
- [ ] Actionable recommendations
- [ ] Proofread

### Delivery
- [ ] Report generated
- [ ] Client presentation ready
- [ ] Raw data archived

---

## Post-Engagement

### Cleanup
- [ ] No tools left on target
- [ ] Test accounts removed
- [ ] Data cleaned up

### Follow-up
- [ ] Questions answered
- [ ] Re-testing scheduled (if needed)
- [ ] Lessons learned documented
