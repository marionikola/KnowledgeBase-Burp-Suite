## General Shortcuts

### Navigation
| Shortcut | Action |
|----------|--------|
| `Ctrl + Tab` | Next tab |
| `Ctrl + Shift + Tab` | Previous tab |
| `Ctrl + 1-9` | Switch to tab 1-9 |
| `Ctrl + W` | Close current tab |
| `Ctrl + F` | Find in current view |
| `Ctrl + L` | Focus URL bar |

### File Operations
| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | New tab |
| `Ctrl + O` | Open project |
| `Ctrl + S` | Save project |
| `Ctrl + Shift + S` | Save project as |
| `Ctrl + Q` | Quit Burp Suite |

### Edit Operations
| Shortcut | Action |
|----------|--------|
| `Ctrl + C` | Copy |
| `Ctrl + V` | Paste |
| `Ctrl + X` | Cut |
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + A` | Select all |

## Customization

### Creating Custom Shortcuts
```
1. Go to Settings > Keyboard Shortcuts
2. Search for command
3. Press desired key combination
4. Save
```

### Useful Custom Shortcuts
```bash
# Recommended custom shortcuts
Ctrl+Shift+R: Send to Repeater
Ctrl+Shift+I: Send to Intruder  
Ctrl+Shift+S: Save all tabs
Ctrl+Shift+N: New blank tab
```
